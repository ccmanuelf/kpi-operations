## Archive

Historical documents from earlier development phases are in `docs/archive/`, organized by subdirectory:

- `archive/iterations/` -- Phase implementation reports and deployment summaries
- `archive/audits/` -- Historical audit reports (security, testing, database, architecture)
- `archive/sprints/` -- Sprint completion reports
- `archive/analysis/` -- Requirements analysis, data field mapping, inference requirements
- `archive/project-planning/` -- Original project planning documents
